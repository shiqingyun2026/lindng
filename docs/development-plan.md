## 阶段一：基础框架与权限

### 1.1 项目初始化与公共组件

* 初始化小程序项目结构，配置 app.json（底部 TabBar：首页、我的）

* 封装网络请求（utils/request.js），统一 baseURL、错误处理

* 公共工具函数（utils/util.js）

* 全局样式（app.wxss）

* 创建首页和我的页面基础文件



### 1.2 用户协议与隐私政策展示

* 独立页面/弹窗，展示用户协议和隐私政策

* 用户必须勾选同意后才能进入授权流程

* 首次打开小程序时显示



### 1.3 微信授权登录 + 手机号授权

* 实现微信登录（wx.login + wx.getUserProfile）

* 手机号授权（wx.getPhoneNumber）

* 将用户信息存储到本地/全局

* 同时提前处理定位等权限申请（wx.getLocation），避免后续开发时再处理

* 处理定位授权拒绝时的降级方案



### 2.2 课程详情页

5. 子女管理菜单增加占位提示。

6. 客服弹窗组件提前到阶段二之前，方便详情页直接使用。
